# Deep3DShooter

A 3D horror shooter game built with Godot 4.

## Features
- FPS player controls (WASD + mouse)
- Zombie enemies with chase AI and melee attacks
- Target dummies with individual body part hitboxes (head/body/arms/legs)
- Headshot damage multiplier (2x), limb damage reduction (0.5x)
- Floating health bars above enemies
- Red star particle hit effects
- Celebration effect every 3 kills (50 stars, double burst)
- Screen damage vignette on player hit
- Grid-based enemy spawner with auto-respawn

## Controls
- WASD: Move
- Mouse: Look around
- Left Click: Shoot
- Escape: Pause/cursor
